# Personal Expense Tracker

A simple frontend web application developed using HTML, CSS, JavaScript, and Local Storage to manage personal income and expenses.

## Project Overview

The Expense Tracker helps users manage their daily finances by recording income and expense transactions. The application automatically calculates the total income, total expenses, and current balance.

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

## Features Implemented

- Add income transactions
- Add expense transactions
- Display transaction history
- Calculate total income
- Calculate total expenses
- Calculate current balance
- Delete transactions
- Store data using Local Storage
- Responsive and user-friendly interface

## Steps to Run the Project

1. Clone the repository

```bash
git clone <repository-link>
```

2. Open the project folder

```bash
cd expense-tracker
```

3. Open `index.html` in any web browser

4. Use VS Code Live Server

